#  My Portfolio

This is my personal developer portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS**, and animated using **framer-motion**. It showcases my work, skills, and offers a way to get in touch with me.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: GSAP (GreenSock Animation Platform)

## Features

- Responsive and modern UI
- Smooth framer-motion animations
- Projects showcase
- Contact via email

## Installation

```bash
git clone https://github.com/mugabodannyshafi/mugabo.git
cd mugabo
pnpm install
pnpm dev
```
